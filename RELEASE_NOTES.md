## Getting Started

### Prerequisites

- VS Code or Cursor
- Node.js 18+
- Git
- Claude Code CLI (`npm install -g @anthropic-ai/claude-code`)
- Anthropic API key (provided during training or bring your own)
- GitHub account (free)

### Quick Start

```bash
# Clone this repo
git clone https://github.com/AmanProjects/TWTAI.git
cd TWTAI

# Install sample project dependencies
cd sample-project
npm install
npm run build
```
